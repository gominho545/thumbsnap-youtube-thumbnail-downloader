# ThumbSnap - Free YouTube Thumbnail Downloader

ThumbSnap is a free online tool to download YouTube thumbnails in HD instantly.  
No login or installation required — just paste the video link and get the thumbnail in seconds.

---

## Features

- Download thumbnails in **HD, SD, and other resolutions**  
- Works instantly from any **YouTube video URL**  
- **No login** or software installation required  
- Supports multiple languages  
- Mobile and desktop friendly  

---

## How to Use

1. Copy the **YouTube video URL**  
2. Paste it in **ThumbSnap**: [https://thumbsnap.pages.dev/youtube-thumbnail-download](https://thumbsnap.pages.dev/youtube-thumbnail-download)  
3. Click **Extract**  
4. Choose your desired resolution and **Download**
