# ThumbSnap - Free YouTube Thumbnail Downloader

ThumbSnap is a simple online tool to download YouTube thumbnails in HD.  
It works directly from the video URL, no login required, and extracts all available resolutions.  

## Features

- Download HD, SD, and other thumbnail sizes  
- Works instantly from any YouTube video URL  
- No software installation required
